Lecture one:

There are two main types of Learning

  • Supervised Learning

    In these kind of problems,we will predict correct answers from given set of data.the problems can be generally classified as two kinds

    • Regression Plot

      In this kind we will plot a continuous graph and the solutions will be continuous

    • classification plot

      in this kind the solutions will be discreet

    In both these cases there can be many variables

  • Unsupervised learning

    These are Seperation Problems.We are given a bunch of unlabeled data

      Examples
    • Google news :Given a bunch of articles without saying anything about them.The site clusters them into particular stories
    • Cocktail party .In which two voices are mixed and the algorithm seperates them into two different kinds
    • Clustering Genomes of people.

    In all of these cases we are not labelling the data as anything.

Lecture Two:

Given a bunch of data with labels.The learning algorithm takes the data and produces a function called Hypothesis:h(x)

where x1,x2,x3...Xm are the inputs and y1,y2,y3...Ym are outputs.h(x)takes inputs x1,x2.. to produce outputs y1,y2...

we can make h(x) as a straight line.then we call this algorithm or method as Linear Regression. and on single variable x ,we can call this as univariate linear regression.
